AMIA Installation Guide

This guide provides step-by-step instructions to install and set up the AMIA workflow on your system.

1. Download the AMIA Workflow

Clone the repository from GitHub:

git clone https://github.com/kbrown3687524/amia

2. Set Up the Conda Virtual Environment

Create and activate a Conda environment using the provided environment file:

conda env create -f amia_environment.yml
conda activate amia

3. Install FoldX

Download and extract FoldX (version 4.0) into the main AMIA directory. Ensure the directory structure looks like this:

|-- AMIA Folder:
  |  
  |-- foldx:
      |-- foldx_4
      |-- yasaraPlugin.zip
      |-- rotabase.txt

4. Test the Installation

Run the provided test dataset to verify the installation:

./AMIA.py --pdb_file ~/test/HIV-1C_ZA/RLT_Model_Repair.pdb --mutations ~/test/HIV-1C_ZA/mutations.csv --output_dir ~/test/variant_outputs

5. File Formats

Ensure the input files are in the correct formats:

  • Structure File: Protein Data Bank (.pdb)
  • Mutations File: Comma Separated Variable (.csv)
